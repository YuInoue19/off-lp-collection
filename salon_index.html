よくあるご質問

導入前の疑問にお答えします

サンプルは本当に無料ですか？ +
はい、無料です。送料も当社負担でお届けします。まずはご自身の手で使用感をお確かめください。
最小何本から注文できますか？ +
既製品は少数本からご注文いただけます。1人サロンのお客様にも無理のない数量設定です。（※詳細はお問い合わせください）
施術用と物販用は別の商品ですか？ +
処方は同じです。容量とパッケージが異なります。施術で使ったものと同じ商品を物販としてご案内いただけるため、お客様への説明もスムーズです。
敏感肌のお客様にも使えますか？ +
安全性試験を実施しておりますが、すべての方に刺激が起きないことを保証するものではありません。施術前のパッチテストをおすすめしています。
OEMの最小ロットはいくつですか？ +
小ロットからお受けしています。ラベルデザインや容器の選定もサポートいたします。（※詳細はお問い合わせください）
卸価格はいくらですか？ +
導入量に応じた価格表をご用意しています。サンプル請求時または導入相談時にお伝えいたします。
他のサロンと同じ商品になりませんか？ +
既製品の場合は同一商品ですが、OEM/PBであれば完全にオリジナル商品をお作りいただけます。「まずは既製品で試す → OEMで独自ブランド化」という段階的な導入も多くいただいています。
商材の仕入れは初めてなのですが大丈夫ですか？ +
はい、初めての方にも安心してお取り組みいただけるよう、メニュー設計や物販の見せ方までサポートしています。お気軽にご相談ください。
